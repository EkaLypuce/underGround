# underGround

petit jeu utilisant node.js (socket.io) et Vue.js (il est fortement recommendé que les joueurs puissent se parler de vives voix)

Règles : 

Le maitre du jeu crée un lobby, les différents joueur rejoignent la partie.
Un mot est attribué à chaques joueurs que seul le joueur pourra voir, mais il y a un intru. Le but est de trouver l'intru, pour se faire,
chaques joueurs écrit deux mots en rapport avec son mot que tout le monde va voir. A vous de faire les bonnes déduction en fonctions des
mots entré par les autres.

Bonne chance, et que le meilleur gagne ...


PS : ceci est la première version, la seconde arrive très vite !
